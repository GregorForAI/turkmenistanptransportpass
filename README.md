# Turkmenistan P-Transport Pass

Проект, связанный с управлением или информацией о транспортных пропусках в Туркменистане.

## Технологии
*   Vite
*   TypeScript
*   React
*   shadcn-ui
*   Tailwind CSS

## Установка и запуск
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/GregorForAI/turkmenistanptransportpass.git
    cd turkmenistanptransportpass
    ```
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите сервер для разработки:
    ```bash
    npm run dev
    ```
